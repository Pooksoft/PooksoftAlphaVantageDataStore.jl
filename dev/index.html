Introduction

This package encodes a set of Julia functions to download data from the AlphaVantage financial data application programming interface (API). This package assumes you have login credentials for the AlphaVantage API.

Installation Requirements

PooksoftAlphaVantageDataStore.jl can be installed, updated, or removed using the Julia package management system. To access the package management interface, open the Julia REPL, and start the package mode by pressing ]. While in package mode, to install PooksoftAlphaVantageDataStore.jl, issue the command:

(@v1.6) pkg> add PooksoftAlphaVantageDataStore

Required packages will be downloaded and installed for you automatically.

To use PooksoftAlphaVantageDataStore.jl in your projects issue the command:

julia> using PooksoftAlphaVantageDataStore

Package Organization

Functions of PooksoftAlphaVantageDataStore.jl are organized around each of the category areas of the AlphaVantage application programming interface. The current version of PooksoftAlphaVantageDataStore.jl implements functions that wrap the Stock Time Series (STS) methods of AlphaVantage.
